# Weather-DashBoard

🌦️ Real-Time Weather Dashboard (Power BI / API-Based Project)
📌 Overview

This project is a real-time weather dashboard that fetches live weather data using API integration and visualizes it through an interactive dashboard. It demonstrates how external APIs can be integrated with data visualization tools to provide up-to-date insights.

The dashboard enables users to monitor real-time weather conditions such as temperature, humidity, wind speed, and other key metrics.

🎥 Project Reference

https://youtu.be/P8HB8dMfKNc?si=OT5ERXc_108Yrm4c

🎯 Objective
Fetch real-time weather data using API calls
Transform and visualize live data in a dashboard
Build an interactive and user-friendly weather monitoring system

🛠️ Tech Stack
Power BI
REST API (Weather API)
Power Query (API integration & transformation)
DAX (for calculated metrics, if applicable)

🔌 API Integration
Connected to a weather API to fetch real-time data
Used Power Query to call API endpoints
Parsed JSON response into structured tabular format
Refreshed data dynamically for live updates

📊 Key Features
Real-time weather data visualization
Temperature, humidity, wind speed tracking
Dynamic dashboard updates
Interactive filters and visuals
Clean and intuitive UI/UX design

🔄 Workflow
Connect to Weather API
Fetch data using API request (JSON format)
Transform data using Power Query
Create calculated fields (if required)
Build dashboard visuals
Enable refresh for real-time updates
